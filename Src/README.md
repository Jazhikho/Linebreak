## Src Module Overview

- `Core`: Application bootstrap, dependency wiring, domain models, and shared services.
- `UI`: Terminal rendering, command shell, and user interaction flows.
- `Simulation`: Systems that model network behaviour, anomalies, and state progression.
- `Conspiracies`: Narrative logic, branching scenarios, and data interpretations.
- `Commands`: Implementations of console commands (`trace`, `decode`, etc.).
- `Saves`: Persistence layer, save-slot management, and SQLite integration.
- `Data`: Static JSON assets, localisation tables, and configuration schemas.
